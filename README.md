 Troubleshooting
 ### Camera Not Working
 **Problem**: Black screen or no camera feed
 **Solutions**:
 1. Check browser permissions (Settings ‚Üí Privacy)
 2. Close other apps using camera
 3. Refresh the page
 4. Try different browser
 5. Restart device
### BPM Not Detecting
 **Problem**: BPM shows 0 or wrong number
 **Solutions**:
 1. Turn up music volume
 2. Use external speaker (not headphones)
 3. Check microphone permission
 4. Try "Tap Sync" button instead
 5. Ensure music has clear beat
 ### Motion Sensors Not Working
 **Problem**: Steps/turns stay at 0
 **Solutions**:
 1. **iOS**: Grant motion permission when prompted
 2. Check if using HTTPS (required)
 3. Try "Calibrate Sensors" in settings
 4. Hold phone or place on stable surface
 5. Actually move around! 
ÔøΩ
ÔøΩ
 ### App Won't Install
 **Problem**: No install option
 **Solutions**:
 1. Use supported browser (Chrome/Safari)
 2. Ensure using HTTPS URL
 3. Clear browser cache
 4. Try in private/incognito mode
 5. Update browser to latest version
 ### Data Not Saving
 **Problem**: Sessions disappear
 **Solutions**:
 1. Don't use private/incognito mode
 2. Check browser storage settings
 3. Ensure not clearing cache automatically
 4. Export data regularly as backup
 ### App is Slow
 **Problem**: Laggy or stuttering
 **Solutions**:
 1. Close other apps
 2. Lower camera quality (Settings)
 3. Clear browser cache
4. Restart app
 5. Update to latest version--
